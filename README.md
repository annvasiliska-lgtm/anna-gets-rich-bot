# Expense Telegram Bot

Простой Telegram-бот для учёта ежедневных расходов.

## Что умеет

- записывать траты сообщением вроде `кофе 35000`
- записывать несколько трат одним сообщением, по одной на строку
- показывать сумму за сегодня командой `/today`
- показывать последние траты командой `/last`
- удалять последнюю трату командой `/undo`
- очищать сегодняшние тестовые записи командой `/clear_today`

## Локальный запуск

1. Создайте бота в Telegram через `@BotFather`.
2. Скопируйте токен.
3. Создайте файл `.env` по примеру `.env.example`.
4. Установите зависимости:

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

5. Запустите бота:

```bash
python bot.py
```

На Mac можно запускать бота двойным кликом по файлу `start.command` или через приложение `Anna Gets Rich Bot.app`.

## Формат расходов

```text
кофе 35000
такси 120000
продукты 450000
```

Сумма должна быть последним числом в сообщении.

## Railway

Для деплоя на Railway нужно добавить переменную окружения:

```text
BOT_TOKEN=ваш_токен_бота
```

Если используете Railway Volume для сохранения базы между деплоями, добавьте:

```text
DATA_DIR=/data
```
